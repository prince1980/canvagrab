# CanvaGrab — Canva Video Downloader

Free tool to download all videos from any Canva portfolio or website page.

## Deploy to Railway (One Click)

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/new)

1. Click the button above (or go to [railway.app](https://railway.app))
2. Sign in with GitHub
3. Select this repo → Deploy

That's it. Railway will build and deploy automatically.

## Run Locally

```bash
npm install
npx playwright install firefox
node server.js
```

Open http://localhost:3000
