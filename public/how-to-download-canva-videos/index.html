Why You Can't Simply Right-Click and Save Canva Videos

Canva portfolio pages and Canva website pages deliver video content using DASH adaptive streaming. Unlike a simple .mp4 file linked on a webpage, Canva splits video and audio into separate encrypted streams. This means:

  • Right-clicking the video and choosing "Save" only downloads the video without any audio
  • Browser dev tools show dozens of fragmented network requests — not a single clean download link
  • Generic video downloaders and browser extensions don't understand Canva's URL structure

The Easiest Way: Use CanvaGrab

CanvaGrab is a free web tool that automates the entire process. Here's how to use it:

  1. Go to the Canva portfolio page or Canva website you want to download from
  2. Copy the full URL from your browser address bar
  3. Open CanvaGrab.veronix.co and paste the URL
  4. Click Download All and wait 15–30 seconds
  5. Download each video as a clean MP4 file with audio

Supported Canva Page Types

  • Canva portfolio pages — yourname.my.canva.site
  • Canva website pages — any custom domain connected to Canva
  • Canva design previews — public canva.com/design/... URLs
